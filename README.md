# 여름방학 자바 스터디

## 기간

7월 1일(월) ~ 8월 31일(토), 총 9주차

## 시간 및 장소

매주 수요일 18시 30분, 오프라인 또는 디스코드

## 스터디원

- 김수완
- 김용래
- 김윤진
- 방현우(스터디장)
- 박소연
- 최승윤

## 활동 내용

- 매주 학습한 내용을 마크다운 형식으로 정리하여 **화요일 23시 59분**까지 레포지토리에 PR합니다.
  - 마크다운 파일명은 `X주차_김XX.md`으로 통일합니다.
- 정기모임 시에 본인이 정리한 내용을 간단하게 발표합니다.
- 추가적으로, 매주 3명씩 특정 주제에 대해 조사 및 발표합니다.
